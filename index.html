科学上网（WireGuard篇）

发表于 2020-07-31 更新于 2019-11-09 分类于 科学上网
本文字数： 287 阅读时长 ≈ 1 分钟

下载WireGuard

请去这里下载与自己系统匹配的WireGuard。

公钥信息

安装完成后打开WireGuard，按Ctrl+N组合键，出现创建新的通道窗口，记住这里的公钥信息。

添加节点

去这里添加节点信息，把前面的公钥填到”Your wireguard public key:”栏，”Your cryptostorm token (or it’s SHA512 hash):”栏可以自己随意敲入字母数字的组合，按 ADD KEY，把生成的信息拷贝粘贴到前面WireGuard下创建新通道的窗口中，注意保留原来窗口中的信息，把重复的栏目删除。

激活

激活后就可以科学上网了。

以下我们以IAR下的C开发为例，聊一聊如何进行具体的单元测试工作。

Hexo+Next+Github搭建个人博客

发表于 2020-07-31 更新于 2019-09-06 分类于 hexo
本文字数： 3.3k 阅读时长 ≈ 3 分钟

本地环境搭建

本文作者基于windws 10系统下进行本地环境的搭建。

安装git

点此选择相应的版本下载并安装。

安装Node.js

点此选择相应的版本下载并安装。

安装确认

找到gitbash，点击进入控制台窗口。

$ git version #查看git版本信息

$ node -v #查看Node.js版本信息

$ npm -v #查看包管理工具版本信息

安装hexo

npm install -g hexo-cli

初始化

在gitbash下，切换到你准备存放博客的目录

$ hexo init my-blog

执行以上操作将会花费一些时间，完成后将会在所在的目录下创建my-blog文件夹，所有本地的文件都会存放在这个文件夹下面。

然后执行以下操作：

$ cd my-blog

$ npm install

本地预览

在完成了本地初始化后，如果没有任何报错，那么可以在gitbash控制台窗口中执行

$ hexo g

$ hexo s

之后显示如下信息：

INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

在网络浏览器的地址栏中输入http://localhost:4000

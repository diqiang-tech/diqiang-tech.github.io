Hexo+Next+Github搭建个人博客

发表于 2023-04-16 分类于 hexo

本地环境搭建

本文作者基于windws 10系统下进行本地环境的搭建。

依赖库
package.json

安装git

点此选择相应的版本下载并安装。

安装Node.js

点此选择相应的版本下载并安装。

安装确认

找到gitbash，点击进入控制台窗口。

$ git version #查看git版本信息

$ node -v #查看Node.js版本信息

$ npm -v #查看包管理工具版本信息

安装hexo

npm install -g hexo-cli

初始化

在gitbash下，切换到你准备存放博客的目录

$ hexo init my-blog

执行以上操作将会花费一些时间，完成后将会在所在的目录下创建my-blog文件夹，所有本地的文件都会存放在这个文件夹下面。

然后执行以下操作：

$ cd my-blog

$ npm install

本地预览

在完成了本地初始化后，如果没有任何报错，那么可以在gitbash控制台窗口中执行

$ hexo g

$ hexo s

之后显示如下信息：

INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

在网络浏览器的地址栏中输入http://localhost:4000

Design Pattern （Rust Edition）
